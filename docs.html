DOCUMENTATION

Hier findest du kurze Dokumentationen sowie Anleitungen

GNS3

GNS3 ist eine Open-Source-Software für
Netzwerksimulationen, die es
Fachleuten ermöglicht, virtuelle
Netzwerke zu erstellen und zu testen.

NGINX

Nginx ist eine Open-Source-Software
für Webserver, die es Profis erlaubt,
Websites und Anwendungen
effizient zu hosten und zu managen.

DOCKER

Docker ist eine Open-Source-Plattform,
die Profis ermöglicht, Anwendungen
in leichten Containern zu entwickeln,
zu deployen und zu managen.

KUBERNETES

Kubernetes ist eine Open-Source-Platt-
form für die Automatisierung,
Bereitstellung und Skalierung
containerisierter Anwendungen.

GITHUB

GitHub ist eine Plattform für
die gemeinsame Entwicklung von
Softwareprojekten, auf der Entwickler
Code hosten und verwalten können.

LINUX

Linux ist ein Betriebssystemkernel,
Basis für verschiedene Open-Source-
Betriebssysteme, anpassbar
von Nutzern.

MICROSOFT TEAMS

Microsoft Teams ist eine Plattform für die
Zusammenarbeit in Echtzeit, die
Chat, Videokonferenzen, und
Dateiaustausch integriert.

BITWARDEN

Bitwarden ist ein Open-Source-Passw-
ord-Manager, der sicheres
Speichern und Verwalten von
Passwörtern ermöglicht.

JAVA LNG

Java ist eine vielseitige Program-
miersprache für plattformunabhän-
gige Softwareentwicklung mit breiter
Anwendung im IT-Bereich.
