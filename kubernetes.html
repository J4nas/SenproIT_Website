KUBERNETES Definition

Kubernetes ist eine Open-Source-Plattform für die Automatisierung
der Bereitstellung, Skalierung und Verwaltung containerisierter
Anwendungen. Es ermöglicht die effiziente Orchestrierung von
Containern über mehrere Hosts hinweg und bietet Funktionen wie
Lastenausgleich, Selbstheilung und deklarative Konfiguration.
Kubernetes erleichtert die Entwicklung und Verwaltung von
Anwendungen in containerisierter Umgebung.

KUBERNETES Installation Windows

1. KUBERNETES herunterladen

Lade Minikube von der offiziellen Website herunter und installiere es.

2. Starte Minikube

"minikube start"

3. Überprüfe den Status des Clusters

"kubectl cluster-info"

4. Erstelle und starte eine einfache Anwendung

"kubectl create deployment hello-world --image=gcr.io/google-samples/hello-app:1.0"
