LINUX Definition

Linux ist ein quelloffener Betriebssystemkernel, der die
Grundlage für verschiedene Distributionen bildet.
Es zeichnet sich durch Stabilität, Sicherheit und
Anpassungsfähigkeit aus. Linux wird in einer Vielzahl
von Umgebungen eingesetzt, von Servern bis zu
Embedded-Systemen, und basiert auf den Prinzipien
von Unix. Es wird von einer globalen Gemeinschaft
freiwilliger Entwickler unterstützt und
kontinuierlich verbessert.

LINUX Installation Ubuntu

1. Aktualisieren des Paketmanagers

"sudo apt-get update"

2. Installieren von Updates

"sudo apt-get upgrade"

3. Installieren von Standard-Entwicklungstools

"sudo apt-get install build-essential"
